反转链表

反转链表

题目来源

  • 时间复杂度：O(n)

  • 空间复杂度 O（1）

  • 图解如下：

  • JS 代码如下:
var reverseList = function(head) {
    let pre = null;
    let cur = head;
    while(cur != null){
        //记录next
        let next = cur.next;
        cur.next = pre;
        pre = cur;
        cur = next;
    }
    return pre;
};
